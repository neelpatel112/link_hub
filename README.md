# LinkHub Frontend

Frontend for LinkHub - A Linktree clone application.

## 🚀 Live Demo
- Backend API: https://linkhub-lxbu.onrender.com
- Frontend: https://linkhub-tan.vercel.app/

## ✨ Features
- User authentication (register/login)
- Create and manage links
- Customizable profile
- Public profile pages
- Responsive design

## 🛠️ Built With
- React
- Tailwind CSS
- React Router
- Axios

## 🏃‍♂️ Run Locally
1. Clone the repository
2. Install dependencies: `npm install`
3. Create `.env` file with `VITE_API_URL=your_backend_url`
4. Run: `npm run dev`
